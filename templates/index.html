¿Qué hace esta aplicación?

Esta aplicación web usa el modelo final seleccionado en mi proyecto de regresión. El objetivo es estimar {{ project.target }}, que representa la temperatura máxima del día siguiente, medida en grados Celsius.

Los datos que se capturan en el formulario son variables originales del dataset, por ejemplo temperatura actual, temperatura estimada por LDAPS y nubosidad. El usuario no tiene que escribir variables transformadas, valores escalados ni componentes de PCA.
